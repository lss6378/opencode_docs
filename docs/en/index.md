## Quick Start

::: code-group

```bash [Using npm]
npm install -g opencode-ai
```

```bash [Using curl]
curl -fsSL https://opencode.ai/install | bash
```

```bash [Using Homebrew]
brew install anomalyco/tap/opencode
```

:::

After installation, run `opencode` to start using it.
