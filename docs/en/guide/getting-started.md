# Getting Started

This guide will help you configure OpenCode and complete your first conversation. It takes approximately **3 minutes**.

## Configure API Keys

### Using OpenCode Zen (Recommended)

1. Run OpenCode: `opencode`
2. Type `/connect`
3. Select `opencode` provider
4. Visit [opencode.ai/auth](https://opencode.ai/auth) to authenticate
5. Add billing info and copy API Key
6. Paste API Key

## Initialize Project

```bash
cd /path/to/your/project
opencode
/init
```

## Basic Commands

| Command | Function |
|---------|---------|
| `/help` | Show help |
| `/init` | Initialize project |
| `/models` | List models |
| `/undo` | Undo changes |
| `/redo` | Redo changes |
| `/new` | New session |
| `/share` | Share conversation |
| `/exit` | Exit |
