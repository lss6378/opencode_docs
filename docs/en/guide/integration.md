# Integration

This guide introduces the main ways to integrate OpenCode. It takes approximately **2 minutes**.

## Terminal User Interface (TUI)

```bash
opencode                    # Start in current directory
opencode /path/to/project    # Start in specified directory
```

## IDE Extensions

- **VS Code**: Search for "OpenCode" extension
- **Cursor**: Built-in OpenCode support
- **Others**: VSCodium, Windsurf, Zed

## Web Interface

```bash
opencode web    # Start Web interface
```

Runs by default at `http://localhost:4096`.

## Desktop Application

Visit [opencode.ai/download](https://opencode.ai/download) to download.

## CLI Mode

```bash
opencode run "Create a new API endpoint"
```
