# Tutorial Navigation

Welcome to the OpenCode quick start tutorial! This tutorial is designed for experienced developers to help you master the core features of OpenCode in 15 minutes.

## Tutorial Path

We recommend following this order:

### 1️⃣ Installation Guide (2 minutes)

Learn how to install OpenCode on different operating systems.

- Prerequisites
- Multiple installation methods
- Verify installation

**[Start Installation →](/en/guide/installation)**

### 2️⃣ Getting Started (3 minutes)

Configure OpenCode and complete your first conversation.

- Configure API Keys
- Initialize project
- First conversation experience

**[Quick Start →](/en/guide/getting-started)**

### 3️⃣ Basic Usage (3 minutes)

Master OpenCode's core functionality and workflow.

- Plan/Build mode switching

**[Learn Basic Usage →](/en/guide/basic-usage)**

### 4️⃣ Advanced Features (4 minutes)

Explore OpenCode's advanced features.

- Multi-session management
- Share conversations

- Custom configuration

**[Explore Advanced Features →](/en/guide/advanced-features)**

### 5️⃣ Integration (2 minutes)

Learn how to use OpenCode in different development environments.

- Terminal usage (TUI)
- IDE extensions
- Web interface
- Desktop application

**[Learn Integration →](/en/guide/integration)**

### 6️⃣ Best Practices (2 minutes)

Learn tips and habits to improve development efficiency.

- Git integration
- Permission control
- Theme and keyboard shortcuts
- Common workflow examples

**[View Best Practices →](/en/guide/best-practices)**
