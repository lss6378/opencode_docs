# Installation Guide

This guide will help you install OpenCode on different platforms. The entire process takes approximately **2 minutes**.

## Prerequisites

- Modern terminal emulator (such as [WezTerm](https://wezterm.org), [Alacritty](https://alacritty.org))
- API Keys for LLM providers (or use OpenCode Zen)

## Quick Installation

### Using Installation Script (Recommended)

```bash
curl -fsSL https://opencode.ai/install | bash
```

### Using NPM

```bash
npm install -g opencode-ai
```

### Using Homebrew

```bash
brew install anomalyco/tap/opencode
```

## Verify Installation

```bash
opencode --version
```

## Next Steps

After successful installation, continue reading [Getting Started](/en/guide/getting-started) to configure and start using OpenCode.
