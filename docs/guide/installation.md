# 安装指南

本指南将帮助您在不同平台上安装 OpenCode，整个过程大约需要 **2 分钟**。

## 前置要求

- 现代终端模拟器（如 [WezTerm](https://wezterm.org)、[Alacritty](https://alacritty.org)）
- 用于 LLM 提供商的 API Keys（或使用 OpenCode Zen）

## 快速安装

### 使用安装脚本（推荐）

```bash
curl -fsSL https://opencode.ai/install | bash
```

### 使用 NPM

```bash
npm install -g opencode-ai
```

### 使用 Homebrew

```bash
brew install anomalyco/tap/opencode
```

## 验证安装

```bash
opencode --version
```

## 下一步

安装成功后，继续阅读[快速入门](/guide/getting-started)来配置并开始使用 OpenCode。
