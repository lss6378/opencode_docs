# 教程导航

欢迎使用 OpenCode 入门教程！本教程专为有经验的开发者设计，帮助您在 15 分钟内掌握 OpenCode 的核心功能。

## 教程路径

建议按照以下顺序学习：

### 1️⃣ 安装指南（2 分钟）

了解如何在不同操作系统上安装 OpenCode。

- 前置要求
- 多种安装方式
- 验证安装是否成功

**[开始安装 →](/guide/installation)**

### 2️⃣ 快速入门（3 分钟）

配置 OpenCode 并完成第一次对话。

- 配置 API Keys
- 初始化项目
- 第一次对话体验

**[快速入门 →](/guide/getting-started)**

### 3️⃣ 基本使用（3 分钟）

掌握 OpenCode 的核心功能和工作流程。

- 文件引用（`@` 符号）
- 运行 Bash 命令（`!` 符号）
- 常用命令
- Plan/Build 模式切换

**[学习基本使用 →](/guide/basic-usage)**

### 4️⃣ 高级功能（4 分钟）

探索 OpenCode 的进阶功能。

- 多会话管理
- 分享对话
- 模型选择与切换
- 自定义配置

**[探索高级功能 →](/guide/advanced-features)**

### 5️⃣ 集成场景（2 分钟）

了解如何在不同的开发环境中使用 OpenCode。

- 终端使用（TUI）
- IDE 扩展
- Web 界面
- 桌面应用

**[了解集成场景 →](/guide/integration)**

### 6️⃣ 最佳实践（2 分钟）

学习提高开发效率的技巧和习惯。

- Git 集成
- 权限控制
- 主题和快捷键配置
- 常见工作流示例

**[查看最佳实践 →](/guide/best-practices)**
