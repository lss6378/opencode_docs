# 快速入门

本指南将帮助您配置 OpenCode 并完成第一次对话，大约需要 **3 分钟**。

## 配置 API Keys

### 使用 OpenCode Zen（推荐）

1. 运行 OpenCode：`opencode`
2. 输入 `/connect`
3. 选择 `opencode` 提供商
4. 访问 [opencode.ai/auth](https://opencode.ai/auth) 认证
5. 添加账单信息并复制 API Key
6. 粘贴 API Key

## 初始化项目

```bash
cd /path/to/your/project
opencode
/init
```

## 第一次对话

```
请简要介绍一下这个项目的结构。
```

## 基本命令

| 命令 | 功能 |
|------|------|
| `/help` | 显示帮助 |
| `/init` | 初始化项目 |
| `/models` | 列出模型 |
| `/undo` | 撤销更改 |
| `/redo` | 重做更改 |
| `/new` | 新建会话 |
| `/share` | 分享对话 |
| `/exit` | 退出 |
