# 集成场景

本指南介绍 OpenCode 的主要集成方式，大约需要 **2 分钟**。

## 终端用户界面（TUI）

```bash
opencode                    # 在当前目录启动
opencode /path/to/project    # 在指定目录启动
```

## IDE 扩展

- **VS Code**: 搜索 "OpenCode" 扩展
- **Cursor**: 内置 OpenCode 支持
- **其他**: VSCodium、Windsurf、Zed

## Web 界面

```bash
opencode web    # 启动 Web 界面
```

默认在 `http://localhost:4096` 运行。

## 桌面应用

访问 [opencode.ai/download](https://opencode.ai/download) 下载。

## CLI 模式

```bash
opencode run "Create a new API endpoint"
```
